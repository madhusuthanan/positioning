float columns

For variable height columns, absolute positioning does not work, so let's come up with another solution. We can "float" an element to push it as far as possible to the right or to the left, and allow text to wrap around it. This is typically used for images, but we will use it for more complex layout tasks (because it's the only tool we have).

If we float one column to the left, then also float the second column to the left, they will push up against each other.
